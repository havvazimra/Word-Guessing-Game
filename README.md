Word Guessing Game

📌 Overview

The Word Guessing Game is a fun and interactive web-based game built using Flask. Players attempt to guess a hidden word by selecting letters before running out of attempts. The game includes multiple difficulty levels, hints, and a scoring system to enhance the experience.

🎮 Features

🏆 Multiple Difficulty Levels: Easy, Medium, and Hard.

💡 Hint System: Get hints to assist in guessing.

⏳ Timer Functionality: Race against time for an added challenge.

📊 Leaderboard: Track high scores and compete with friends.

🎨 Responsive UI: Dark-themed design for an immersive experience.

🔧 Installation

Prerequisites:

Python 3.7+ installed on your system.

Setup Guide:

Clone the repository

Navigate to the project folder:

cd Word-Guessing-Game

Create and activate a virtual environment (optional but recommended):

python -m venv venv
# Activate venv
source venv/bin/activate   # On macOS/Linux
venv\Scripts\activate     # On Windows

Install dependencies:

pip install -r requirements.txt

Run the Flask application:

python app.py

Open the game in your browser:
Navigate to http://127.0.0.1:5000/
